Java I/O流(输入/输出流)

IO：Java对数据的操作是通过流的方式，IO流用来处理设备之间的数据传输，上传文件和下载文件，Java用于操作流的对象都在IO包中。

InputStream 和 OutputStream 两个类都是抽象类，抽象类不能进行实例化。在实际应用中我们要用到一系列基本数据流类，都是他们的子类，在实现其父类方法的同时又都定义了其特有的功能。
